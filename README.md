This is a template project for IS2S564 Web Content Management Technologies
The project can be cloned and built with docker-compose.
Modify the docker-compose.yml file to set the location for your files, this ensures you store your files permanently, and can transfer the project between computers.

Once you are ready to launch, run
```bash
docker-compose up
```
in your terminal, then visit
```bash
localhost:8080
```
in your web browser to setup. Note, the port number 8080 can be changed in the docker-compose file.
